# RJI Konsultan Indonesia - Company Profile Website

Website company profile profesional untuk RJI Konsultan Indonesia yang menyediakan layanan konsultasi bisnis.

## Fitur

- **Desain Modern & Responsif**: Tampilan yang menarik dan optimal di semua perangkat
- **Navigasi Smooth**: Navigasi yang halus dengan smooth scrolling
- **Animasi Interaktif**: Efek hover dan animasi scroll untuk pengalaman pengguna yang lebih baik
- **Formulir Kontak**: Formulir kontak yang mudah digunakan
- **Mobile Friendly**: Menu mobile yang responsif
- **SEO Optimized**: Meta tags dan struktur HTML yang SEO-friendly

## Teknologi yang Digunakan

- **HTML5**: Struktur website
- **TailwindCSS**: Framework CSS untuk styling modern
- **JavaScript**: Interaktivitas dan animasi
- **Font Awesome**: Icon library
- **Google Fonts (Inter)**: Typography yang modern dan clean

## Struktur Website

1. **Beranda**: Hero section dengan call-to-action
2. **Tentang Kami**: Informasi tentang perusahaan dan statistik
3. **Layanan**: 6 layanan utama yang ditawarkan
   - Konsultasi Strategi Bisnis
   - Manajemen SDM
   - Optimasi Proses Bisnis
   - Konsultasi Keuangan
   - Transformasi Digital
   - Manajemen Risiko
4. **Keunggulan**: 4 keunggulan utama perusahaan
5. **Kontak**: Formulir kontak dan informasi kontak
6. **Footer**: Navigasi tambahan dan newsletter

## Cara Menggunakan

1. Buka file `index.html` di browser Anda
2. Website siap digunakan tanpa instalasi tambahan

## Cara Deploy

### Hosting Statis (Netlify, Vercel, GitHub Pages)

1. Upload semua file ke repository Git
2. Connect repository ke platform hosting pilihan Anda
3. Deploy otomatis akan berjalan

### Manual Upload

1. Upload file `index.html` dan `script.js` ke web hosting Anda
2. Pastikan struktur folder tetap sama
3. Website siap diakses

## Kustomisasi

### Mengubah Warna

Edit bagian gradient di `index.html`:

```css
.gradient-bg {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### Mengubah Konten

Edit teks di file `index.html` sesuai kebutuhan Anda.

### Menambah/Mengurangi Layanan

Edit section `#layanan` di `index.html` untuk menambah atau mengurangi card layanan.
